# 🤖 AI Resume Screening and Parsing System

## 📌 Overview
This project is an AI-powered Resume Screening and Parsing System that automates the process of extracting, analyzing, and evaluating candidate information from resumes. It uses machine learning and natural language processing (NLP) techniques to assist HR professionals in making faster and more accurate hiring decisions.

The system helps reduce manual effort, improves efficiency, and ensures a structured approach to recruitment.

---

## 🎯 Objectives
- Automate resume parsing and data extraction
- Screen candidates based on relevant criteria
- Reduce manual workload in recruitment
- Improve accuracy and efficiency in hiring
- Enable data-driven decision making

---

## 🚀 Features
- 📄 Resume parsing (extract skills, education, experience)
- 🤖 AI/ML-based candidate screening
- 🔍 Keyword-based filtering
- 📊 Candidate evaluation and ranking
- ⚡ Fast and efficient processing

---

## 🛠️ Tools & Technologies Used
- Python
- Google Colab (for development and execution)
- Pandas & NumPy (data processing)
- Scikit-learn (machine learning)
- Natural Language Processing (NLP)
